# IPL Data Pipeline

An end-to-end ETL data pipeline built using Python, Pandas, and SQLite.

## What this project does
- Extracts IPL match and ball-by-ball data from CSV files
- Cleans and transforms data using Pandas
- Loads cleaned data into a SQLite database
- Analyzes data using SQL queries to generate cricket insights

## Tech Stack
- Python
- Pandas
- SQLite
- Jupyter Notebook

## Key Insights
- Top run scorers and wicket takers across all IPL seasons
- Toss impact on match results
- Team win percentages
- Highest scoring overs analysis

## Dataset
IPL Complete Dataset from Kaggle (2008–2024)
